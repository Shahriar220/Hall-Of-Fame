Hall of Fame Vidz

Hall of fame is a django app to make custom youtube playlist. Made with django and sqlite3. The login,signup and logout are generated from default django login and signup views. All the forms are generated by django forms. Decorators were added to protect the views from other users.

In add video the form submission is done by ajax. 

In the views.py file search_video views is used to search the video data from the api. To fetch the youtube api data django request module is used.  Ajax was used to communicate with the server without reloading the page.
